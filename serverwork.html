We run Windows Server 2012 R2 (I know) on a machine that's been EOL for the past > half decade (I know, alright?). It's fine - at least we don't use crowdstrike. Here, I host a list of all the group policies and various settings I use on our system. Updated whenever I have the time (once every few months)

The Group Policies are divided by purpose, as well as whether they're a computer or user policy.

There's also explanations for the policies and some of their meanings.



The Setup

The institution has 3-5 classrooms, each with 7-10 computers. This includes student computers and 1 instructor computer per class. There's also a few computers in the front - for the front desk, to display information, to show things on the TV while students wait for their class, etc. There's also a workshop - a sort-of classroom with less restriction for the computers. Additionally, each student, teacher, admin staff, etc. that works/studies at the institution has a personal account with their own (standardised) credentials. Furthermore, in the server, each account has it's own folder, which is only accessible to that account (through a home drive connection) or if explicitly searched for.

The server also has shared drives for software, common files, file sharing, etc.

Finally, the server has some additional features - for instance, it's possible to use PXE Boot to reinstall Windows on all the computers from one centralised image. This lowers the time spent doing so for the entire center from ~1 day to ~30 minutes plus change (assuming all the computers behave - if they don't it comes out to ~30-40 minutes per computer).
